how to add YOUR specific photos & videos

📸 Add your own pictures: Scroll down in the code to the photoSet array (around line 310). Replace the image URLs with your own image links (use free hosting like Imgur, Google Drive shared links, or local paths if hosting).

🎬 Fix / customize videos: Find videoSet array (around line 320). Replace the video URLs with your own video files (upload to cloud or use your own .mp4 links). I've already added fully functional video examples from Pixabay (safe & working).

💡 Pro tip: To use local images/videos, put them in a folder named "assets" and use paths like "assets/photo1.jpg" or "assets/video.mp4". The carousels will automatically update.
